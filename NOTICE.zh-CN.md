# 说明

[中文](NOTICE.zh-CN.md) | [English](NOTICE.md)

这个仓库是一个聚焦型辅助层，不是对上游项目的完整替代，也不是更大私有工作区的打包导出。

它的公开范围被刻意限制在以下几件事：

- 让手机安全访问本地 Codex 会话
- 让新手机首次登录时经过可信设备审批
- 为单用户、自托管场景提供部署和 hardening 指南

本仓库建立在上游 `siteboon/claudecodeui` 基础上。

- 上游项目：`siteboon/claudecodeui`
- 适配测试版本：`v1.25.2`
- 上游许可证：`GPL-3.0`

推荐理解方式是：

1. 先在本地准备上游源码
2. 再应用本仓库的覆盖层
3. 最后按照本仓库里的部署与安全文档完成运行

公开发布时，应只保留最小必要的可复用材料：覆盖层、辅助脚本、桌面控制工具源码，以及部署 / 安全 / 贡献 / 发版审查文档。

以下内容不应该和这个开源仓库一起公开发布：

- 个人数据库
- 运行日志
- 诊断导出包
- 证书和私钥
- 打包好的二进制
- 维护者内部说明或一次性发布材料
- 父目录工作区中的 sibling project 或其它私人资产
- 超出最小必要范围的完整上游源码快照
- 真实域名、私有 IP、审批痕迹和设备运行证据

如果你要公开发布自己的 fork，请保留上游归属和许可证说明，并先完成本地运行痕迹的脱敏。
